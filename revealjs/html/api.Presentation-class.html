             Presentation

Class for the presentatoin. This is the outermost container for all of the presentation's objects. Its childredn should all be Page instances.

The html code is compiled by putting together the contents of a header html file, the copmiled html code of the childrem and a footer html file. The header and footer are string templates. header takes one parameter: the inline CSS style tag. footer takes 3 template parameters: width(int), height(int), theme(str)
